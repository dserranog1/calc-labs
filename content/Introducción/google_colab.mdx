# 2. Google Colab

## ¿Por qué Google Colab?

Existen múltiples formas de programar en Python. Por ejemplo:

- **Instalando Python localmente** y ejecutando archivos `.py` desde la terminal o un editor de código.
- Usando entornos de desarrollo como **VS Code**, **PyCharm**, o **Sublime**
- A través de **notebooks interactivos**, que permiten combinar código, texto, gráficos y resultados en un mismo documento.

Durante estos laboratorios, utilizaremos **Google Colab**, una plataforma que implementa notebooks en la nube.

---

## Ventajas de Google Colab

- **No requiere instalación**: funciona directamente desde el navegador.
- **Acceso inmediato**: solo se necesita una cuenta de Google.
- **Integración con Google Drive**: los notebooks se guardan automáticamente y se pueden compartir fácilmente.
- **Soporte para bibliotecas científicas** como `numpy`, `matplotlib`, `sympy` y muchas más.
- **Entorno interactivo**: permite ejecutar código en pequeñas secciones, ver resultados inmediatos, graficar funciones, y documentar el proceso.

Estas características hacen que los notebooks sean especialmente útiles para **explorar ideas, visualizar conceptos matemáticos y documentar procedimientos paso a paso**, lo cual es ideal en un entorno de aprendizaje como este curso.

---

## Acceder a Google Colab

1. Abran [https://colab.research.google.com](https://colab.research.google.com)
2. Inicien sesión con su cuenta de Google.
3. Seleccionen “Archivo” > “Nuevo cuaderno”.

Pueden nombrar su notebook haciendo clic en el nombre por defecto (“Untitled0.ipynb”) en la esquina superior izquierda.

---

## Estructura de un notebook

Un notebook está compuesto por celdas. Existen principalmente dos tipos:

- **Celdas de código**: donde se escribe y ejecuta código Python.
- **Celdas de texto**: donde se escribe documentación, fórmulas y explicaciones usando un lenguaje de marcado llamado _Markdown_.

Para ejecutar una celda de código, se puede:

- Presionar `Shift + Enter`
- Hacer clic en el botón de ▶️ junto a la celda

---

## Primeros pasos prácticos

1. Crear una nueva celda de código y escribir:

```python
2 + 2
```

2. Ejecutar la celda. El resultado debe aparecer justo debajo.

3. Luego probar el siguiente comando:

```python
print("Hello world!")
```

Con esto confirmamos que el entorno está funcionando correctamente y que el notebook puede ejecutar código Python de forma interactiva.
