## ðŸ§ª Ejercicio: aproximar con Simpson y comparar

Vamos a aplicar el mÃ©todo de Simpson compuesto para aproximar el valor de la siguiente integral:

$$
\int_0^2 \frac{1}{1 + x^2} \, dx
$$

Esta integral tiene un valor exacto conocido:

$$
\int_0^2 \frac{1}{1 + x^2} \, dx = \arctan(2) \approx 1.1071487
$$

### Â¿QuÃ© se va a hacer?

3. Comparar con el valor entregado por `scipy.integrate.quad`.
4. Graficar cÃ³mo evoluciona el error porcentual al aumentar $n$.


## âœ… SoluciÃ³n

```python
import numpy as np
import matplotlib.pyplot as plt
from sympy import symbols, integrate, atan
from scipy.integrate import quad

# FunciÃ³n a integrar
def f(x):
    return 1 / (1 + x**2)

# ImplementaciÃ³n del mÃ©todo de Simpson
def simpson(f, a, b, n):
    if n % 2 != 0:
        raise ValueError("n debe ser par.")
    h = (b - a) / n
    suma = f(a) + f(b)
    for i in range(1, n):
        x_i = a + i * h
        if i % 2 == 0:
            suma += 2 * f(x_i)
        else:
            suma += 4 * f(x_i)
    return (h / 3) * suma
```
