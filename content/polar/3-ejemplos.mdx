#  Ejemplos concretos de puntos

Ahora que sabemos c贸mo convertir entre coordenadas cartesianas y polares, practiquemos con algunos ejemplos representativos.



---

###  Ejemplo 1: De polares a cartesianas

Supongamos el punto en coordenadas polares:

```math
(r, \theta) = (2, \frac{\pi}{3})
```

Aplicamos las f贸rmulas:

```math
x = r \cos(\theta) = 2 \cos\left(\frac{\pi}{3}\right) = 2 \cdot \frac{1}{2} = 1
\\
y = r \sin(\theta) = 2 \sin\left(\frac{\pi}{3}\right) = 2 \cdot \frac{\sqrt{3}}{2} = \sqrt{3}
```

Entonces, el punto cartesiano es $(1, \sqrt{3})$.

---

###  Ejemplo 2: De cartesianas a polares

Ahora partimos del punto:

```math
(x, y) = (-\sqrt{3}, -1)
```

Calculamos:

```math
r = \sqrt{x^2 + y^2} = \sqrt{3 + 1} = 2
\\
\theta = \text{atan2}(-1, -\sqrt{3}) = \frac{5\pi}{3} \text{ (o } -\frac{\pi}{3} \text{)}
```

Entonces, el punto polar es $(2, 5\pi/3)$.

        theta = punto["theta"]
        x = r * np.cos(theta)
        y = r * np.sin(theta)
        label = f"(r={r:.1f}, 胃={theta:.2f})"
    else:

        y = punto["y"]
