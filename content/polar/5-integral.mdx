# Integrales en coordenadas pol√°res

## ‚úçÔ∏è ¬øDe d√≥nde viene la f√≥rmula del √°rea en coordenadas polares?

Cuando trabajamos en coordenadas cartesianas, calculamos el √°rea bajo una curva $y = f(x)$ con:

```math
A = \int_a^b f(x)\, dx
```

Esto funciona porque estamos sumando **tiras verticales** de altura $f(x)$ y base $dx$.

---

### üìê ¬øC√≥mo cambiamos esta idea en coordenadas polares?

En coordenadas polares, no sumamos rect√°ngulos, sino **sectores circulares muy delgados**.

Imaginen que dividimos una regi√≥n curva (como una flor o un c√≠rculo) en muchos **"triangulitos redondeados"**, como las rebanadas de una pizza üçï.  
Cada rebanada tiene:

- Un radio $r = r(\theta)$
- Un peque√±o √°ngulo $d\theta$

El √°rea de un sector circular es:

```math
A = \frac{1}{2} r^2 \theta
```

Entonces, si $\theta$ es muy peque√±o ($d\theta$), podemos usar esta f√≥rmula para aproximar el √°rea **de una rebanada infinitesimal**:

```math
dA = \frac{1}{2} r^2(\theta) \, d\theta
```

---

### üîÑ Sumando infinitas rebanadas‚Ä¶

Para obtener el √°rea total entre los √°ngulos $\theta_1$ y $\theta_2$, simplemente integramos:

```math
A = \int_{\theta_1}^{\theta_2} \frac{1}{2} \left[ r(\theta) \right]^2\, d\theta
```

Esta es la f√≥rmula del √°rea en coordenadas polares

---

### üé® Visualizaci√≥n: sectores que construyen un √°rea

El siguiente c√≥digo muestra c√≥mo una curva en polares puede ser aproximada por varios sectores, cada uno con su propio radio y √°ngulo.

```python
import numpy as np
import matplotlib.pyplot as plt

# Par√°metros para la curva
theta_full = np.linspace(0, np.pi, 1000)
r_full = 1 + 0.5 * np.sin(2 * theta_full)

# Convertimos a coordenadas cartesianas
x_full = r_full * np.cos(theta_full)
y_full = r_full * np.sin(theta_full)

# Sectores aproximados (pocos para visualizaci√≥n)
theta_sect = np.linspace(0, np.pi, 8)
r_sect = 1 + 0.5 * np.sin(2 * theta_sect)

fig, ax = plt.subplots(figsize=(6, 6))
ax.set_aspect('equal')
ax.set_xlim(-2, 2)
ax.set_ylim(-0.5, 2.5)
ax.set_title("Aproximaci√≥n del √°rea con sectores vs. curva real")
ax.grid(True)

# √Årea total encerrada por la curva (relleno suave)
ax.fill(x_full, y_full, alpha=0.2, color='pink', label='√Årea real')

# Curva real
ax.plot(x_full, y_full, color='crimson', label=r'$r = 1 + 0.5\sin(2\theta)$')

# Sectores circulares aproximados
for i in range(len(theta_sect)-1):
    Œ∏1 = theta_sect[i]
    Œ∏2 = theta_sect[i+1]
    r = r_sect[i]
    t = np.linspace(Œ∏1, Œ∏2, 30)
    rt = np.full_like(t, r)
    x = rt * np.cos(t)
    y = rt * np.sin(t)
    ax.fill(np.concatenate([[0], x, [0]]), np.concatenate([[0], y, [0]]), alpha=0.3, color='orange')
    ax.plot(x, y, color='darkorange')

# Centro
ax.plot(0, 0, 'ko')
ax.legend()
plt.show()
```

## üßÆ Ejemplo: √°rea encerrada por $r = 1 + \cos(\theta)$

Queremos calcular el **√°rea encerrada** por la curva polar:

```math
r(\theta) = 1 + \cos(\theta)
```

Esta curva es una **cardioide m√°s estilizada**, con forma de gota o coraz√≥n. Es un ejemplo cl√°sico y visualmente claro de curva que **no puede expresarse f√°cilmente en coordenadas cartesianas**.

---

### üìç Paso 1: Visualizar la curva

```python
import numpy as np
import matplotlib.pyplot as plt

theta = np.linspace(0, 2*np.pi, 1000)
r = 1 + np.cos(theta)
x = r * np.cos(theta)
y = r * np.sin(theta)

plt.figure(figsize=(6, 6))
plt.plot(x, y, color='darkred', label=r'$r = 1 + \cos(\theta)$')
plt.fill(x, y, alpha=0.2, color='darkred')
plt.gca().set_aspect('equal')
plt.title("Curva polar: $r = 1 + \cos(\\theta)$")
plt.grid(True)
plt.legend()
plt.show()
```

---

### üìç Paso 2: Determinar los l√≠mites de integraci√≥n

La curva se **cierra completamente** cuando $\theta$ recorre desde $0$ hasta $2\pi$, as√≠ que:

```math
\theta \in [0, 2\pi]
```

---

### üìç Paso 3: Aplicar la f√≥rmula del √°rea

```math
A = \int_0^{2\pi} \frac{1}{2} \left[1 + \cos(\theta)\right]^2 \, d\theta
```

Expandimos el integrando:

```math
(1 + \cos(\theta))^2 = 1 + 2\cos(\theta) + \cos^2(\theta)
```

Y usamos la identidad:

```math
\cos^2(\theta) = \frac{1 + \cos(2\theta)}{2}
```

Entonces:

```math
A = \frac{1}{2} \int_0^{2\pi} \left(1 + 2\cos(\theta) + \frac{1 + \cos(2\theta)}{2}\right) \, d\theta
\\
= \frac{1}{2} \int_0^{2\pi} \left(\frac{3}{2} + 2\cos(\theta) + \frac{1}{2}\cos(2\theta)\right) \, d\theta
```

---

### üìê Integramos t√©rmino a t√©rmino:

```math
A = \frac{1}{2} \left[
\frac{3}{2} \cdot 2\pi + 2 \cdot \int_0^{2\pi} \cos(\theta)\, d\theta + \frac{1}{2} \cdot \int_0^{2\pi} \cos(2\theta)\, d\theta
\right]
\\
= \frac{1}{2} \left[3\pi + 0 + 0\right] = \frac{3\pi}{2}
```

---

### ‚úÖ Resultado final

El √°rea encerrada por la curva $r = 1 + \cos(\theta)$ es:

```math
A = \frac{3\pi}{2}
```

---

### üß™ Verificaci√≥n con SymPy

```python
from sympy import symbols, integrate, cos, pi, simplify

Œ∏ = symbols('Œ∏')
r = 1 + cos(Œ∏)
area_expr = 0.5 * r**2
area = integrate(area_expr, (Œ∏, 0, 2*pi))
simplify(area)
```

---

### üß† Conclusiones del ejemplo

- Esta curva tiene **forma reconocible y no trivial**, lo que la hace ideal para estudiar integrales en coordenadas polares.
- La integral es manejable usando identidades trigonom√©tricas.
- El √°rea se obtiene sin convertir a coordenadas cartesianas.
