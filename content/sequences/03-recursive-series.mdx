### üß™ Ejemplo 1: Sucesi√≥n cuadr√°tica desplazada

Recordemos esta sucesi√≥n:

```math
2,\ 5,\ 10,\ 17,\ 26,\ \ldots
```

La regla recursiva es:

$$
a_1 = 2, \quad a_n = a_{n-1} + 2n - 1
$$

Implementaci√≥n en Python:

```python
def sucesion_cuadratica(n):
    if n == 1:
        return 2
    else:
        return sucesion_cuadratica(n - 1) + 2 * n - 1

# Mostrar los primeros 10 t√©rminos
for i in range(1, 11):
    print(sucesion_cuadratica(i))
```

### üß™ Ejemplo 2: Sucesi√≥n recursiva con dependencia doble

Consideremos ahora una sucesi√≥n definida como:

$$
a_1 = 1,\quad a_2 = 3,\quad a_n = a_{n-1} + a_{n-2} + 1 \quad \text{para } n \geq 3
$$

```python
def sucesion_doble(n):
    if n == 1:
        return 1
    elif n == 2:
        return 3
    else:
        return sucesion_doble(n - 1) + sucesion_doble(n - 2) + 1

# Mostrar los primeros 10 t√©rminos
for i in range(1, 11):
    print(sucesion_doble(i))
```

### üìà Visualizaci√≥n de una sucesi√≥n

Podemos usar `matplotlib` para representar gr√°ficamente los primeros t√©rminos y observar el tipo de crecimiento.

```python
import matplotlib.pyplot as plt

def generar_sucesion(f, n_max):
    return [f(n) for n in range(1, n_max + 1)]

terminos = generar_sucesion(sucesion_cuadratica, 15)

plt.plot(range(1, 16), terminos, marker='o')
plt.title("Sucesi√≥n cuadr√°tica desplazada")
plt.xlabel("n")
plt.ylabel("a_n")
plt.grid(True)
plt.show()
```
