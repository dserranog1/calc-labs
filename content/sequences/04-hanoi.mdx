# Torres de Hanoi y crecimiento exponencial


### üóº El problema

El juego de las Torres de Hanoi consiste en mover una pila de discos desde un poste inicial a otro, con las siguientes reglas:

1. Solo se puede mover un disco a la vez.
2. No se puede colocar un disco grande sobre uno m√°s peque√±o.
3. Solo se puede mover el disco que est√© arriba de una torre.

Dado un n√∫mero $n$ de discos, ¬øcu√°l es la secuencia m√≠nima de movimientos necesarios para resolver el problema?

---

### üîÅ Soluci√≥n recursiva

La idea recursiva es:

- Mover los $n - 1$ discos superiores a un poste auxiliar.
- Mover el disco m√°s grande al destino.
- Mover los $n - 1$ discos del auxiliar al destino.

Este procedimiento se repite para cada subconjunto de discos.

```python
def hanoi(n, origen, destino, auxiliar):
    if n == 1:
        print(f"Mover disco 1 de {origen} a {destino}")
    else:
        hanoi(n - 1, origen, auxiliar, destino)
        print(f"Mover disco {n} de {origen} a {destino}")
        hanoi(n - 1, auxiliar, destino, origen)

# Ejecutar para 3 discos
hanoi(3, 'A', 'C', 'B')
```

---

### üìà Sucesi√≥n asociada

Llamemos $T_n$ al n√∫mero m√≠nimo de movimientos necesarios para resolver el problema con $n$ discos. A partir del algoritmo recursivo, podemos deducir la siguiente relaci√≥n:

```math
T_1 = 1,\quad T_n = 2T_{n-1} + 1
```

Desarrollando los primeros t√©rminos:

```math
T_2 = 2T_1 + 1 = 3 \\
T_3 = 2T_2 + 1 = 7 \\
T_4 = 2T_3 + 1 = 15 \\
T_5 = 2T_4 + 1 = 31
```

Se puede inducir que la f√≥rmula cerrada es:

```math
T_n = 2^n - 1
```

Este crecimiento es **exponencial**: al aumentar $n$, el n√∫mero de pasos se duplica y se suma uno.

### üìä Visualizaci√≥n del crecimiento

```python
import matplotlib.pyplot as plt

def movimientos(n):
    return 2**n - 1

n_vals = list(range(1, 21))
t_vals = [movimientos(n) for n in n_vals]

plt.plot(n_vals, t_vals, marker='o')
plt.title("N√∫mero de movimientos en Torres de Hanoi")
plt.xlabel("n (n√∫mero de discos)")
plt.ylabel("T(n) = 2‚Åø - 1")
plt.yscale('log')  # opcional: para ver el crecimiento exponencial como l√≠nea recta
plt.grid(True)
plt.show()
```

### üí° Conclusiones

- Las Torres de Hanoi son un ejemplo cl√°sico de c√≥mo **la recursi√≥n resuelve un problema estructurado**.
- El n√∫mero de pasos forma una sucesi√≥n recursiva con f√≥rmula cerrada: $T_n = 2^n - 1$.
- Este tipo de crecimiento aparece en muchos algoritmos y estructuras de datos.
